ハチドリのドット絵をここに置くと、Proodos（index.html）の左下キャラに差し替わります。

【ファイル名（必須）】
  bird-01.png  … ① まばたき
  bird-02.png  … ② 羽上げ
  bird-03.png  … ③ 羽下げ
  bird-04.png  … ④ ついばむ
  bird-05.png  … ⑤ 首かしげ（左）
  bird-06.png  … ⑥ 首かしげ（右）
  bird-07.png  … ⑦ あくび
  bird-08.png  … ⑧ 横向き
  bird-09.png  … ⑨ 上見る
  bird-10.png  … ⑩ 飛ぶ①
  bird-11.png  … ⑪ 飛ぶ②
  bird-12.png  … ⑫ ぴょん
  bird-13.png  … ⑬ 毛づくろい

  ＜撫でられ差分（クリックでランダム表示・複数OK）＞
  bird-14.png  … 撫でられ その1
  bird-15.png  … 撫でられ その2
  bird-16.png  … 撫でられ その3
  bird-17.png  … 撫でられ その4
  bird-18.png  … 撫でられ その5
  （bird-24.png まで増やせます。ある分だけ自動で使われます）

【仕様】
  - 透過PNG、正方形推奨（64×64 など。大きくてもOK、表示時に縮小されます）
  - 同じサイズだと並びが安定します
  - bird-01〜bird-13 が「1分ごと／タブ操作」でランダム表示（通常ポーズ）
  - bird-14〜bird-24 が「クリックで撫でられ」表示。複数あればその中からランダム
    （撫でると2.6秒後に通常ポーズへ戻る）
  - 番号が飛んでいてもOK（存在する画像だけを自動検出して使います）

【置いたら】
  index.html を開き直すだけ。画像が読めればドット絵に自動で差し替わり、
  読めなければ内蔵の簡易ドット絵にフォールバックします（壊れません）。

【GitHubで公開する場合】
  この assets フォルダごとリポジトリに含めてください（相対パスで参照しています）。
